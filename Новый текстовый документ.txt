IF иструкция 
используется в тех случиях когда код должен выполниться только при определенных условиях 
при использовании if else идет преобразование в буллевый тип
else создает возможность дополнительного варианта результата, если основной код иструкции вернет ложь
IF ELSE используется для того чтобы выполнть сразу несколько условий
